# Frontend Blazor (em migração)

Este diretório marca a transição do frontend para **Blazor WebAssembly**. A UI atual em React permanece funcional, mas a entrega final deve priorizar Blazor.

## Objetivo
- Reaproveitar o backend .NET existente.
- Unificar stack com C#/.NET para reduzir custo de manutenção.
- Entregar uma UI consistente com o domínio do projeto.

## Próximos passos sugeridos
1. Criar o projeto Blazor WebAssembly.
2. Replicar as telas essenciais: Login, Dashboard, Despesas, UBS, Relatórios.
3. Integrar autenticação JWT com o backend .NET.
4. Validar fluxo mínimo (login → listar despesas → relatório).

## Observações
- Este diretório é um marcador da migração solicitada.
- A documentação principal será ajustada para refletir o foco Blazor.
